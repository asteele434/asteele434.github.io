Conditionals

English grammar worksheets and explanations for the first conditional, second conditional, and third conditional. 




First conditional
if + present simple will + base verb*
If I am hungry,  I will buy a burger.
If I am not hungry, I will not buy a burger.
If you make this shot, you will win the game.
If you do not make this shot,  they will win the game.
You will not win the game if you do not make this shot.

if + present simple…will + base verb

When using the first conditional, you use the simple present to talk about something that will happen in the future. Click here for first conditional worksheets and to learn more about the first conditional.





Second conditional
if + past simple would + base verb*

If I won the lottery,

I would quit my job.

If I knew,

I would tell you.

If I were president,

I would lower taxes.

I would lower taxes

if I were president.

if + past simple simple…would have + base verb

When using the second conditional, we are discussing unreal, unlikely, or imaginary situations. Click here for second conditional worksheets and to learn more about the second conditional.





Third conditional
if + past perfect would have + past participle

If I had played,

I would have won.

If you had applied,

 you would have gotten the job.

If I did not lose my keys,

 I would have been here earlier.

I would have been here earlier

if I did not lose my keys.

if + past perfect…would have + past participle

When using the third conditional, we are talking about past situations that did not happen and imagining what might have occurred. Click here for third conditional worksheets and to learn more about the third conditional.
